# Data Access Examples

This project demonstrates three different data access methods:

1. API call (JSON) via Scryfall
2. Local file read (CSV) with pandas
3. HTML scraping using BeautifulSoup on Wikipedia

## Setup
Clone the repo.

`pip install requests pandas os beautifulsoup4`

## Run
`python data_access_examples.py`

Be sure to run the script in the project root and ensure the .py file and the .csv file are in the same location.
